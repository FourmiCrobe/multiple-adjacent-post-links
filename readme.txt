== Description ==
Helps generate links to multiple previous and next posts.

== Installation ==
1. Unzip the download, and copy the 'multiple-adjacent-post-links' plugin into your WordPress plugins folder, normally located in /wp-content/plugins/. In other words, all files for the plugin should be in wp-content/plugins/multiple-adjacent-post-links/
2. Login to Wordpress Admin and activate the plugin.
3. Add `multiple_next_post_links();` and `multiple_previous_post_links()` to your posts template.
